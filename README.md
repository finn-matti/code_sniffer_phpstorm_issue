# Testproject

- clone this repo
- run `composer install` in the `code-quality` folder
- adjust paths in the `bin/phpcs` and `bin/phpcbf` files
- create project in PHPStorm with the `testProject` folder
- configure Codesniffer using the `bin/phpcs` file
